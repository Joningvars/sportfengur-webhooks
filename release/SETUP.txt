SportFengur Webhooks - Quick Setup (Windows)

1) Copy .env.example -> .env and fill in:
   - EIDFAXI_USERNAME
   - EIDFAXI_PASSWORD
   - CONTROL_AUTH_USERNAME
   - CONTROL_AUTH_PASSWORD
   - PORT (default 3000)
   - (optional) WEBHOOK_SECRET_REQUIRED=true + SPORTFENGUR_WEBHOOK_SECRET

2) Run the app:
   npm install
   npm start

3) Verify:
   http://localhost:3000/health

Webhook header (if enabled):
   x-webhook-secret: <your secret>
